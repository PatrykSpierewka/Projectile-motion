# Projectile-motion

## Initial conditions:
$$
x(0) = 0
$$

$$
y(0) = 0
$$

$$
\frac{{\mathrm{d} x(0)}}{{\mathrm{d} t}} = V_{0x}
$$

$$
\frac{{\mathrm{d} y(0)}}{{\mathrm{d} t}} = V_{0y}
$$

## Model equations:
$$
\frac{{\mathrm{d} x^2(t)}}{{\mathrm{d} t^2}} = 0
$$

$$
\frac{{\mathrm{d} y^2(t)}}{{\mathrm{d} t^2}} = -g
$$
